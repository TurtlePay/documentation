TurtlePay Public API Interface

Base URL: /v1, Version: 0.0.1, TurtlePay GitHub Organization

This document outlines the public API methods available using the TurtlePay platform for creating and completing payment requests on the TurtleCoin network.

application/json

200 OK

Payment Request Created Successfully

sendToAddress: string

The address to provide the user to send funds to

atomicAmount: number

The atomic amount that the user needs to send to complete the request

startHeight: number

The blockchain height that we will start looking for transactions to complete the request

endHeight: number

The blockchain height that we will stop looking for transactions to complete the request

confirmations: number

The number of confirmations that we will wait to pass once we find transactions destined for this request

qrCode: string

A link to a QR code image that contains the required payment information to complete this request

userDefined: object

A copy of the data you supplied in the optional userDefined field
